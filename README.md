# Nutrifit+
Como desplegar este proyecto en local utilizando XAMPP
1. Dentro de la ruta xampp/htdocs, pega la carpeta nutrifit
2. Dirigete a phpmyadmin e importa la base de datos nutrifit.sql
3. Escribe la ruta en el navegador 127.0.0.1/nutrifit
4. Si has hecho todo correctamente, te deberia de salir el login de nutrifit, lo que significaria que has desplegado correctamente el proyecto en local.
